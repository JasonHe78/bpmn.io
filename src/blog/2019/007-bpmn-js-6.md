## Connect From/To Anywhere

As of this release, you’re able to connect elements regardless of what the direction of the resulting connection will be. If the elements can be connected, you’re able to connect them. The same also applies to reconnecting connections to other elements.

## Looking Forward To Feedback

Read through the [full changelog](https://github.com/bpmn-io/bpmn-js/blob/master/CHANGELOG.md#500) to learn about all issues that were addressed with `bpmn-js@6` as well as breaking changes that may require your action during the upgrade.

Did we miss anything? Did you spot a bug or would you like to suggest an improvement? Reach out to us via [our forums](https://forum.bpmn.io), tweet us [@bpmn_io](https://twitter.com/bpmn_io) or file an issue you found in [the bpmn-js issue tracker](https://github.com/bpmn-io/bpmn-js/issues).

Get the latest BPMN toolkit pre-packaged or as source code via [npm](https://www.npmjs.com/package/bpmn-js) or [unpkg](https://unpkg.com/bpmn-js/).
